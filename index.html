PHM 383 Quality Improvement and Implementation Science Quiz

1. What is the primary focus of Quality Improvement?

Correct Answer: 1. Quality Improvement focuses on context-dependent and local solutions to address specific problems.

2. What is the main goal of Implementation Science?

Correct Answer: 2. Implementation Science aims to develop general knowledge about the benefit of interventions.

3. Which framework is associated with Implementation Science?

Correct Answer: 3. CFIR is a framework commonly used in Implementation Science.

4. Which component is part of the COM-B model?

Correct Answer: 4. The COM-B model includes Capability (Physical), Motivation, and Opportunity as its components.

5. Which model is commonly used in Quality Improvement?

Correct Answer: 1. Lean Six Sigma is commonly used in Quality Improvement to enhance efficiency by reducing waste and variability.

6. What is the primary approach used by Quality Improvement?

Correct Answer: 1. Quality Improvement uses rapid design and testing to quickly address specific problems.

7. What does Implementation Science primarily focus on?

Correct Answer: 3. Implementation Science focuses on understanding the general benefits and broader implications of interventions.

8. Which framework is most associated with the COM-B model?

Correct Answer: 2. COM-B is primarily associated with behavior change, addressing factors such as capability, motivation, and opportunity.

9. Which of the following is an element of the Opportunity component in COM-B?

Correct Answer: 1. The Opportunity component includes factors like physical and social environments that facilitate behavior change.

10. Which model includes both Motivation-Automatic and Motivation-Reflective?

Correct Answer: 4. The COM-B model includes both Motivation-Automatic and Motivation-Reflective components.

11. What is a key goal of Quality Improvement?

Correct Answer: 2. Quality Improvement aims to solve specific, context-dependent problems to improve processes.

12. Which of the following is commonly used to reduce waste and variability in Quality Improvement?

Correct Answer: 1. Lean Six Sigma is a commonly used approach in Quality Improvement to reduce waste and variability.

13. Which element is included in the Capability component of COM-B?

Correct Answer: 4. The Capability component of COM-B includes both psychological and physical capabilities.

14. Which framework is NOT typically used in Quality Improvement?

Correct Answer: 3. CFIR is used in Implementation Science, not typically in Quality Improvement.

15. What is the main characteristic of the "Rapid reevaluation" approach in Quality Improvement?

Correct Answer: 2. Rapid reevaluation in Quality Improvement focuses on continuous iteration and improvement.

16. Which framework is used to understand the benefit of an intervention in different contexts?

Correct Answer: 1. CFIR (Consolidated Framework for Implementation Research) is used to understand the benefit of an intervention across different contexts.

17. Which component of COM-B addresses social factors that influence behavior?

Correct Answer: 3. Opportunity - Social addresses social factors that influence behavior within the COM-B model.

18. What is a key focus of Implementation Science compared to Quality Improvement?

Correct Answer: 2. Implementation Science focuses on understanding how effective interventions are across various contexts.

19. Which model primarily focuses on enabling behavior through motivation and opportunity?

Correct Answer: 4. COM-B focuses on enabling behavior change by addressing motivation, capability, and opportunity.

20. Which Quality Improvement approach involves ongoing evaluation and modifications?

Correct Answer: 1. Quality Improvement often involves rapid iteration to make continuous improvements.

21. What is the purpose of the 4E framework?

Correct Answer: 2. The 4E framework is used to understand the factors that determine effective implementation of an intervention.

22. Which model is most closely associated with improving operational efficiency in health care?

23. Which framework is used to determine what works in a specific context for implementation?

Correct Answer: 1. CFIR (Consolidated Framework for Implementation Research) helps determine what works in specific contexts for effective implementation.

24. What type of motivation is included in the COM-B model?

Correct Answer: 4. The COM-B model includes both reflective and automatic motivation as key components.

25. Which element of COM-B is linked to environmental changes facilitating behavior?

Correct Answer: 2. Opportunity in COM-B refers to environmental and social changes that facilitate behavior.

26. What is the evaluation approach in Implementation Science?

Correct Answer: 3. Implementation Science uses methodical design and testing to understand the general benefits of an intervention.

27. Which model is used to change behavior through training and persuasion?

Correct Answer: 4. COM-B uses interventions such as training, persuasion, and environmental changes to influence behavior.

28. What is a key goal of Lean Six Sigma in Quality Improvement?

Correct Answer: 2. Lean Six Sigma focuses on reducing process variability and eliminating waste to improve quality.

29. What does Capability in COM-B refer to?

Correct Answer: 3. Capability in COM-B includes both physical and psychological abilities required for behavior change.

30. Which framework involves understanding different contextual factors affecting implementation success?
